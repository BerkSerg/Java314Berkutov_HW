
Пример кода на Java:

java

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Этот код представляет собой простую программу на Java, которая выводит сообщение "Hello, World!" в консоль.

Использование Java для создания приложений:

Java используется для разработки разнообразных приложений, включая веб-приложения, мобильные приложения, настольные приложения и корпоративные системы. Он также широко используется в области разработки игр, научных и инженерных приложений, финансовых и банковских систем и многих других областях.
